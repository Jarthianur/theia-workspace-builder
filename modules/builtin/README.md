# builtin

This module adds _almost_ all vscode builtin plugins to theia.
It does not provide a fully featured all-in-one workspace, but adds basic language support like in a default vscode installation.

## Features

- clojure
- coffeescript
- cpp
- csharp
- css
- emmet
- fsharp
- go
- groovy
- grunt
- gulp
- handlebars
- hlsl
- html
- jake
- java
- javascript
- less
- lua
- npm
- objective-c
- php
- pug
- python
- r
- razor
- ruby
- rust
- scss
- shaderlab

- typescript
- vb
